# 🛒 Mini E-commerce

Este é um projeto simples de **loja virtual** desenvolvido para fins de estudo e portfólio de desenvolvimento **Front-end**.

O sistema permite visualizar produtos, adicionar itens ao carrinho e finalizar um pedido.  
Os dados do carrinho são armazenados no navegador e os pedidos são enviados para o servidor utilizando PHP.

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- PHP

---

## 📦 Funcionalidades

- Listagem de produtos
- Adicionar produtos ao carrinho
- Armazenamento do carrinho com LocalStorage
- Visualização do total do pedido
- Página de carrinho
- Finalização de pedido (checkout)
- Salvamento dos pedidos no servidor

## ⚙️ Como executar o projeto

1. Clone o repositório:

git clone https://github.com/seuusuario/mini-ecommerce

2. Coloque a pasta do projeto dentro de um servidor local como:

- XAMPP
- WAMP
- Laragon

3. Abra o navegador e acesse:

http://localhost/mini-ecommerce
